# Page photo d'anniversaire

Une page HTML/CSS simple pour afficher des photos depuis un QR code.

## Ajouter les photos

1. Mets tes images dans le dossier `photos`.
2. Renomme-les comme ceci: `photo-01.jpg`, `photo-02.jpg`, `photo-03.jpg`, etc.
3. La page prevoit deja 12 photos. Pour en afficher plus, copie une ligne de photo dans `index.html` et change le numero.

## Modifier le texte

Dans `index.html`, tu peux changer:

- `Joyeux anniversaire`
- le petit texte sous le titre
- le message dans la section `Pour toi`
- la phrase du bas de page

## Publier avec GitHub Pages

1. Cree un depot GitHub.
2. Ajoute ce dossier dans le depot.
3. Dans GitHub, va dans `Settings` > `Pages`.
4. Dans `Build and deployment`, choisis `Deploy from a branch`.
5. Selectionne la branche `main` et le dossier `/root`.
6. GitHub donnera une adresse du type `https://ton-nom.github.io/nom-du-depot/`.

## Creer le QR code

Quand la page est publiee, copie l'adresse GitHub Pages et colle-la dans un generateur de QR code.
